сщ# Ресурсы Duck Rescue

Сейчас в папке лежат локальные изображения:

- `forest-floor.jpg` — текстура покрытия игровой сцены;
- `forest-sky.jpg` — фон для `a-sky`.

## Главный файл, который нужно добавить из Blender

Положите сюда:

- `duck_collectible.glb`
- `fox_hazard.glb`

Именно это имя ожидают:

- `index.html` для игровых уток;
- `ar.html` для AR-сцены на телефоне через камеру и гироскоп.

## Быстрая инструкция экспорта из Blender

 1. Соберите/выберите модели для утки и лисы в Blender (или другой 3D-пакет).
2. Запустите скрипт.
3. Выделите меши капсулы.
4. Выберите `File -> Export -> glTF 2.0`.
5. Формат: `glTF Binary (.glb)`.
6. Имена файлов: `duck_collectible.glb`, `fox_hazard.glb`.
7. Сохраните файл в эту папку.

## Как сцена подхватывает модель

В `index.html` и `ar.html` есть автоматическая проверка:

- если `duck_collectible.glb` найден, загружаются ваши утки; если `fox_hazard.glb` найден, лисы;
- если файла нет, показывается встроенный fallback-объект из примитивов A-Frame.
